## ‚õ≥ About The Game
**Mini Golf 3D** is a physics-based arcade golf game developed in Unity. 

The goal is simple: Aim, shoot, and sink the ball in the hole with the fewest strokes possible. The game features **5 unique levels** ranging from simple straight shots to complex courses with bridges, curved paths, and moving obstacles.

## ‚ú® Key Features
* **5 Challenging Levels:** progressively harder courses to test your putting skills.
* **Classic Obstacles:** Navigate past static blocks, tight bridges, and rotating **Windmills**.
* **Physics Interaction:** Realistic ball movement and collision mechanics.
* **Full UI System:**
    * **Home Page:** Animated 3D Main Menu with a castle theme.
    * **HUD:** Real-time level tracking and FPS display.
* **Post-Processing:** polished visuals with bloom and color correction.
